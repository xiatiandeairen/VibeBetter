# AI 生成代码质量检查清单

每次 Review AI 生成的代码时，按此清单逐项检查。

## 必检项 (Must Check)

1. **类型安全** — 无 `any`、无不必要的 `as` 断言
2. **错误处理** — 使用 AppError，无吞掉的 catch
3. **边界条件** — null/undefined/空数组/0 均有处理
4. **命名语义** — 变量名、函数名表达意图而非实现
5. **测试覆盖** — 新增逻辑有对应单元测试

## 应检项 (Should Check)

6. **重复消除** — 与现有代码无功能重复
7. **文件大小** — 修改后 ≤ 200 行
8. **安全性** — 无 SQL 注入、XSS、敏感信息泄露
9. **性能** — 无 N+1 查询、无不必要的全量加载
10. **依赖** — 无不必要的新外部依赖
