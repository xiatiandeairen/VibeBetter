# v0.11.0 Release Notes — Automated Weekly Digest

Weekly activity summary to increase user engagement.

- `DigestService` — Backend service for weekly digest generation
- `GET /api/v1/metrics/projects/:id/digest` — Weekly digest endpoint
- `vibe digest` — CLI command for weekly summary in terminal
- Shows: AI metrics, PSRI/TDI trends, PR activity, snapshot count
- Period: last 7 days with trend comparison
