# v0.13.0 Release Notes — Actionable Fix Suggestions

Users see risk scores — now they know WHAT to do about them.

- `vibe fix` — Suggests specific actions for high-risk files
- Analyzes: complexity, change frequency, file size, author count, AI ratio
- Suggestions: split modules, reduce complexity, stabilize interfaces, assign ownership
- Priority levels: high (red), medium (yellow), low (dim)
- Includes recovery tips on error (e.g., "Run `vibe status` to check connection")
