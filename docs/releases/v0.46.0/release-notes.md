# v0.46.0 Release Notes — Sparkline History (User)

CLI command showing metric trends as sparkline charts.

- `vibe history` displays PSRI, TDI, and AI Success Rate as sparklines
- Uses Unicode block characters (▁▂▃▄▅▆▇█) for visual trends
- `--count` flag to control number of snapshots
- Shows date range and latest values alongside sparklines
