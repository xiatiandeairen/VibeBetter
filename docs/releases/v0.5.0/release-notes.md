# v0.5.0 Release Notes — AI 归因 + 组织级 + 基准线

从「能用的工具」进化为「有洞察深度的产品」。

- **AI 归因分析**：AI 代码 vs 人工代码的质量对比（复杂度/稳定性/回滚率）
- **组织级 Dashboard**：一个页面看所有项目健康状态
- **开发者效能排行**：按开发者维度分析 AI 使用效果
- **行业基准线**：Excellent/Good/Average/Poor 标签让用户知道自己的位置
- **失败 PR 归因**：大修改/回滚/高 Review 轮数 分类统计
- **GitHub Actions CI**：自动化 lint + test + build
- **文档体系重构**：编号版本归档 + 清晰目录结构
