# v0.8.0 Release Notes — CLI: AI Coding 场景化落地

从「看板」到「行动」：`vibe` CLI 工具将洞察嵌入开发者工作流。

- `vibe check` — 提交前风险检查（核心命令）
- `vibe insights` — AI 成果率 + 工具使用摘要
- `vibe decisions` — 查看/管理决策建议
- `vibe report` — 生成 Markdown/JSON 健康报告
- `vibe risk` — 项目/文件级 PSRI 查询
- `vibe sync` — 触发数据采集 + 指标计算
- `vibe init` / `vibe status` — 配置管理
- API Key 认证，可集成为 git pre-commit hook
